AlignEval

Knowledge Alignment Evaluation • Detection → Evaluation → Optimization

Precisely detect, quantify, and visualize knowledge defects in fine-tuned large language models. Build source & learned knowledge graphs, align them, and identify missing or incorrect knowledge.

1. Upload Dataset

Load domain Q&A pairs. NER & relation extraction build the source knowledge graph.

2. Probe LLM

Multi-level prompts (factual, relational, reverse) probe your model to build the learned knowledge graph.

3. Evaluate

Align source & learned graphs. Compute Precision, Recall, F1 and locate defects.

4. Optimize

Get actionable data / model / reasoning optimization suggestions based on detected gaps.
